# Following packagaes are used to built this application

- Nodemailer
- Body-parser
- Nodemon
- Concurrently
- Express
- Axios
- Express-Handlebars
- React
- React-bootstrap bootstrap
- React-router-dom
- stripe/react-stripe-js
- stripe/stripe-js
- stripe express cors
- dotenv

Please run the following command to install the packages if required;

```
npm install <packages name>
```
